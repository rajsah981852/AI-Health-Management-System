
# HealthMate AI Project Setup Guide

## Prerequisites
- Node.js (version 18 or higher)
- npm or Bun

## Quick Setup Steps

### 1. Clone the Repository
```bash
git clone <YOUR_REPOSITORY_URL>
cd healthmate-ai
```

### 2. Install Dependencies
```bash
# If using npm
npm install

# If using Bun
bun install
```

### 3. Start Development Server
```bash
# If using npm
npm run dev

# If using Bun
bun run dev
```

### 4. Open in VS Code
After cloning, you can open the project directly in VS Code:

```bash
# From terminal
code .

# Or simply open VS Code and use File > Open Folder
```

## Recommended VS Code Extensions
- ESLint
- Prettier
- Tailwind CSS IntelliSense
- TypeScript Vue Plugin (Volar)

## Troubleshooting
- Ensure you have the correct Node.js version
- Run `npm cache clean --force` if dependency issues occur
- Check console for specific error messages

## Project Structure
- `src/`: Main application source code
- `components/`: Reusable UI components
- `pages/`: Application route components
- `hooks/`: Custom React hooks
- `context/`: React context providers
